# DHL (dhl)

DHL is a global logistics company that provides a wide range of services, including express delivery, freight transportation, supply chain solutions, and e-commerce services. As one of the largest logistics companies in the world, DHL operates in over 220 countries and territories. The DHL API Developer Portal at developer.dhl.com exposes a rich catalog of APIs across business divisions including DHL eCommerce, DHL Express, DHL Global Forwarding, DHL Freight, DHL Supply Chain, and Post and Parcel Germany.

## APIs

### Unified APIs

- **Location Finder Unified** - Single interface to discover DHL parcel and letter locations across divisions.
- **Shipment Tracking Unified** - Unified shipment status across DHL Freight, eCommerce, Supply Chain, Global Forwarding, and Post and Parcel Germany.
- **Shipment Tracking Unified Push** - Push-based shipment status updates across DHL divisions.
- **Duty and Tax Calculator Unified** - Carrier-agnostic duty and tax calculator with a 30-day free trial.

### DHL eCommerce Americas

- **User Guide DHL eCommerce Americas** - End-to-end shipping, duty/tax, labels, manifests, and tracking.
- **Authentication DHL eCommerce Americas** - Access token generation.
- **Changelog DHL eCommerce Americas** - Product updates and release notes.
- **Duty and Tax DHL eCommerce Americas** - Cross-border duty and tax calculation.
- **Product Finder DHL eCommerce Americas** - Shipping products, rates, and delivery estimates.
- **References DHL eCommerce Americas** - Reference data, fields, and glossary.
- **Return Label DHL eCommerce Americas** - Domestic return label creation and retrieval.
- **Tracking DHL eCommerce Americas** - Single and bulk package tracking.

### DHL eCommerce Europe

- **DHL eCommerce Europe eConnect** - Cross-border European shipments and returns.
- **DHL eCommerce Europe eConnect Beta** - Beta features for European eCommerce shipping.
- **DHL eCommerce UK** - Domestic and international parcel shipments from the UK.
- **Parcel EU BE LU NL** - Labels for shipments across Belgium, Luxembourg, and the Netherlands.

### DHL eCommerce India (Blue Dart)

- **Pickup Cancellation DHL eCommerce India** - Cancel scheduled pickups.
- **Product and Sub-Product Pickup Detail DHL eCommerce India** - Pickup detail and product codes.
- **Registration for Pickup DHL eCommerce India** - Schedule pickups for shipment orders.
- **Shipment Tracking DHL eCommerce India** - Detailed shipment status for Blue Dart.

### DHL Freight

- **Additional Services DHL Freight** - Validates services that combine with European freight products.
- **Authentication API DHL Freight** - Authentication for freight operations.
- **Price Quote DHL Freight** - Pricing for European road freight.
- **Print DHL Freight** - GS1 and ANSIFACT barcode labels and shipment documents.
- **Product DHL Freight** - Available product codes for European road freight.
- **Shipment Booking DHL Freight** - European palletized road freight transport orders.
- **Time Table DHL Freight** - Product codes and projected delivery dates.

### DHL Global Forwarding

- **Document DHL Global Forwarding** - Download electronic freight documents.
- **Push API DHL Global Forwarding** - Subscription-based push messages for freight shipments.
- **Shipment Booking DHL Global Forwarding** - Air and ocean freight booking.
- **Shipment Label DHL Global Forwarding** - Air and ocean freight labels.
- **Shipment Status DHL Global Forwarding** - Latest shipment events.
- **Shipment Tracking v2 DHL Global Forwarding** - Comprehensive tracking with emissions and routing data.

### Post and Parcel Germany

- **Authentication API Post Parcel Germany** - Authentication for business customers.
- **DATAFACTORY AUTOCOMPLETE 2.0** - Automatic postal data completion for address entry.
- **Deutsche Post Hybrid Mail Shipments E-POST** - Send PDFs as physical mail with tracking.
- **Deutsche Post International** - Labels for international mail and lightweight items.
- **Deutsche Post INTERNETMARKE** - Online postage for domestic and international mail.
- **Deutsche Post Order Management AM** - Electronic order management for commercial logistics.
- **Deutsche Post Print-Mailing Dispatch Preparation** - Dialogue marketing print mailings.
- **Deutsche Post Print-Mailing Targeting** - Plan dialogue marketing campaigns.
- **DHL Parcel DE Pickup** - Place pickup orders and query pickup locations.
- **DHL Parcel DE Postnumber** - Validate postnumbers for parcel locker delivery.
- **DHL Parcel DE Private Shipping** - Create DHL private customer shipments.
- **DHL Parcel DE Returns** - Return label creation across European countries.
- **DHL Parcel DE Shipping** - Business customer shipment label management.
- **DHL Parcel DE Tracking** - Shipment status and history queries.
- **Products API Post Parcel Germany** - Deutsche Post product portfolio for Internetmarke partners.
